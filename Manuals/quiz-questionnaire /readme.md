# Quiz – Questionnaire Module (Team Leader)

This quiz accompanies the “Team Leader – Questionnaire Module” training guide in Valitracks.  
Its purpose is to ensure that users understand the essential steps for preparing and managing questionnaires in the system, including:

- Importing assets (XLIFF, Excel, Word)
- Preferred formats for Linguistic Validation (LV)
- Enabling instruments for ERP, Workflow, LV, and Invivo Testing
- Preparing sections in the source language
- Using the Pencil icon for text edits, interviewer notes, question marking, and numerical mode
- Using the “i” icon to add concept definitions and clarifications for all linguists
