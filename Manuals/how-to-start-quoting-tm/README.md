Valitracks Manuals

This repository contains the interactive training manuals for Valitracks.
Each folder represents one manual exported from Adobe Captivate.

Available manuals:
- Import Word / Excel / XLIFF
- Accept Company Owner Invitation
- How to Start Quoting (Team Leader Module)

The manuals are deployed automatically to:
https://manuals.valitracks.io
