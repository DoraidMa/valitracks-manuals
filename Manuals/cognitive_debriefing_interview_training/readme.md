# Cognitive Debriefing Interview Training Module

This interactive HTML module provides a complete training walkthrough for conducting Cognitive Debriefing (CD) interviews at Valitracks. It explains how interviewers access their tasks, how they prepare informed consent, how to conduct trials, how to review questionnaires, how to add participants, and how to complete final summaries.
