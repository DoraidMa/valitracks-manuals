# How to Prepare the Summary Report on Valitracks

This manual explains how interviewers and clinicians complete the Summary Report inside a Cognitive Debriefing (CD) trial on Valitracks. It covers how to edit section summaries, use the commenting and tasking systems, submit the trial, and complete the Final Summary once enabled by the Project Manager. Autosave is enabled throughout the summary and final summary process.
