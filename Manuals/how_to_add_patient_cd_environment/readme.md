# How to Add a Patient to Your Cognitive Debriefing (CD) Environment

This manual explains how interviewers can add participants (patients) to the Cognitive Debriefing environment in Valitracks. It covers both online and onsite (offline) trials and clarifies when patient information must be manually added by the interviewer.

## File Location

## Contents Covered
- When you need to add a patient manually  
- When participants appear automatically as ready trials  
- How to access the Patients tab  
- How to add a new patient  
- Required and optional fields (name, email, phone number)  
- Choosing between Online vs. On-Site trials  
- Understanding when an email is required  
- How patients appear later in the Trials tab
