# How to Import Internal Contacts

This folder contains the interactive manual for Q3: "How to Import Internal Contacts" in Valitracks.

This module explains:
- Where to add contacts (Company → Contacts)
- How to manually create new contacts using the +New Contact option
- How to assign the correct roles to each contact:
  • Translator
  • Reviewer
  • Reconciler
  • Clinician
  • Interviewer
  • Patient
  • Project Manager
- How to enter pricing and financial information for each role
- How translators must have source/target languages added
- How to add multiple language pairs and multiple pricing variables
- How to bulk-import contacts using the Excel template via "Export Contacts"
- How to upload the completed Excel file to add many contacts at once

This module is part of the Team Leader workflow setup sequence and prepares your company for assigning tasks inside workflows.
