# How to Create a Trial (Step 1)

This manual explains how to create a new Cognitive Debriefing (CD) trial inside the Valitracks Interviewer interface. It covers creating the trial, accessing the trial dashboard, using the messaging and tasking systems, starting the interactive interview, uploading files, filling demographics, reviewing questionnaire sections, and submitting or canceling the trial.
