# How to Review Prices in the Project Offer Module

This manual explains how to use the Review Prices popup after creating or duplicating a Project Offer. 
It describes each field, how to select pricing types, how to apply updated system prices, and the importance of clicking "Save" to commit the changes.

Topics Covered:
- Opening the Review Prices popup
- Field explanations (Current Price, System Price, Pricing Type, Difference)
- Applying prices individually or in bulk
- Saving updated prices
- Handling missing pricing (info icon)
