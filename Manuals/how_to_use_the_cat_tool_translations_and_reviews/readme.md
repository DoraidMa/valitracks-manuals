# How to Use the CAT Tool for Translations and Reviews

This manual explains how linguists can use the Valitracks CAT Tool to perform Translation, Proofreading (Review), Editing, and eCOA Review tasks. It covers how to access tasks, work inside the CAT environment, use comments and notes, AI suggestions, instructions, locks, and how to finalize and deliver the task properly.
