<h2>README</h2>
<p>This guide explains how a Clinician Reviewer should perform a clinical review inside Valitracks. It covers:</p>
<ul>
    <li>How to access the review task through My Pages → My Translations</li>
    <li>Opening the task using the four-squares icon</li>
    <li>Reviewing the translation from a medical and clinical perspective</li>
    <li>Using the purple notebook commenting system to add comments</li>
    <li>Viewing and responding to tasks and comments through the notification system</li>
    <li>Handling PM/QA tasks and marking them as completed</li>
    <li>Using the <strong>“Mark as Ready”</strong> button</li>
    <li>Electronically signing the declaration confirming your medical and language expertise</li>
</ul>
<p>This manual is intended for all clinicians performing reviews as part of the Valitracks linguistic validation workflow.</p>
