# How to Set Up Services for My Company

This folder contains the interactive manual for Q2: "How to Set Up Services for My Company?" in Valitracks.

This module covers:
- How to add services manually inside the Company → Services section
- How each service must be assigned to the correct Service Type
- How service types connect to user roles and workflow creation
- How to create pricing rules for each service
- The 5 pricing types in Valitracks:
  • Minimum Charge
  • Percentage Type
  • Per Word
  • Fixed Price
  • Per Hour
- How to bulk import services using the Excel template

This training module is part of the Company Owner (Team Leader) workflow setup process and prepares the ground for workflow building and quotation automation.
