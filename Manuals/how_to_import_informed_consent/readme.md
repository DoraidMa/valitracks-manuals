# How to Import an Informed Consent

This manual explains how interviewers can import and configure an Informed Consent document inside the Pre-Trial section of the Valitracks Cognitive Debriefing (CD) module. It covers naming the consent action button, adding optional information, and uploading the PDF that the participant will view during the interview.

## Contents Covered
- Accessing the Pre-Trial section in the Interviewer Interface  
- Using the “+ Add Section” button  
- Naming the action button in the participant’s native language  
- Adding optional descriptive text  
- Uploading the Informed Consent file (PDF only)  
- Saving and viewing the imported consent section
